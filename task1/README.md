# Задача 1

## Сборка
- `make` – собирает `task1`
- Поддерживаются переменные `CFLAGS` и `LDFLAGS`

## Очистка
- `make clean` – удаляет `task1` и `*.o`

## Зависимости
- Компилятор gcc
- Библиотека pthread 

## Использование
Программа читает числа из стандартного входа и выводит отсортированный массив

## Пример запуска:
echo 3 2 1 | ./task1 <количество_потоков>
