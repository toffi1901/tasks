# Задача 8 

## Сборка
- `make` – собирает исполняемый файл `task8`
- Поддерживаются переменные `CFLAGS` и `LDFLAGS` 

## Очистка
- `make clean` – удаляет `task8` и объектные файлы (`*.o`)

## Зависимости
- Компилятор `gcc`
- Библиотека `pthread` 
- Требуется поддержка `_GNU_SOURCE` (включена в `CFLAGS`)

## Пример запуска
```bash
./task8 --signal
./task8 --sigaction
