# Задача 2 

## Сборка
- `make` – собирает исполняемый файл `task2`
- Поддерживаются переменные `CFLAGS` и `LDFLAGS` 

## Очистка
- `make clean` – удаляет `task2` и объектные файлы (`*.o`)

## Зависимости
- Компилятор `gcc`
- POSIX-система (Linux/Unix) с поддержкой именованных каналов
