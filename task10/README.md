# Задача 10 

## Сборка
- `make` – собирает исполняемый файл `task10`
- Поддерживаются переменные `CXXFLAGS`

## Очистка
- `make clean` – удаляет `task10` и все объектные файлы

## Зависимости
- Компилятор `g++` 

## Использование
Программа читает из стандартного ввода одну строку с логическим выражением.  
Выводит таблицу истинности: шапка с именами переменных и столбец `Result`, затем строки значений.

## Пример запуска
```bash
echo "A AND B" | ./task10
